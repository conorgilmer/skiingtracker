such as
+ Lifts 
+ Slopes
+ No. Black, Red, Blue etc.
+ Highest point
+ Lowest point
as well as
+ Distance
+ Distance SKi-Km
+ Distance SKiLift-Km
+ Time
+ speed
+ stops
+ free/guiding
